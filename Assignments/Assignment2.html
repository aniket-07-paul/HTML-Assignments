Some common text formatting tags

Tag Description Example
Bold Text (<b>) Used to make text bold <b>This is bold text</b>
Italic Text (<i>) Used to make text italic. <i>This is italic text</i>
Underlined Text (<u>) Used to underline text. <u>his is underlined text</u>
Strong Importance (<strong>) Represents text of strong importance, typically displayed as bold. <strong>This is strongly emphasized text</strong>
Emphasized Text (<em>) Represents text with emphasized meaning, typically displayed as italic. <em>This is emphasized text</em>
